# Habit Tracker

## About The Project
這是我第三個網頁Project,也是我第二個用React寫的網站,與前面兩個Project比較起來,這個Project的CSS有明顯的進步,除了網站有響應式以外,網站的整體GUI設計跟排版也大勝前面兩個Project。
使用這個網站可以幫助用戶紀錄自己想要每天達成的好習慣,並且保存歷史紀錄與製作相關的圖表。

## Getting Started
* npm
  ```sh
  npm install npm@latest -g
  ```
## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/yang0709-dev/Habit-Tracker
   ```
2. cd to the "habit-tracker" folder
   ```sh
   cd habit-tracker/
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm run dev
   ```
